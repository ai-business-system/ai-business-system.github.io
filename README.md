# AiOps & Automation

B2B-портфолио специалиста по автоматизации бизнес-процессов и AI-интеграциям.

## Стек

- Astro (SSG)
- TailwindCSS
- GitHub Pages

## Локальный запуск

```bash
npm install
npm run dev
```

Сайт будет доступен на `http://localhost:4321`

## Сборка

```bash
npm run build
```

Готовый статический сайт будет в директории `dist/`

## Деплой

GitHub Pages автоматически деплоит ветку `gh-pages` 
через GitHub Actions.
